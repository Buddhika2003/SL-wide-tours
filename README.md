# SL Wide Tours Founder Profile Website

A modern and fully responsive business profile website developed for **Darshana Jayawardana**, Founder of **SL Wide Tours**. This website presents the founder's profile, company branding, contact information, awards, and online presence through a clean and professional user interface optimized for all devices.

---

## 🌟 Features

- Fully responsive design
- Mobile, Tablet, and Desktop friendly
- Founder profile showcase
- Clickable phone numbers
- Clickable email addresses
- WhatsApp direct contact integration
- Official website link
- Smooth scrolling navigation
- Professional tourism-themed design
- Award and recognition display
- Optimized image layouts

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome
